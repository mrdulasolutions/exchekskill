# ExChek Encryption (ENC / 5x992) Classification & Notification

Help with **5A992/5D992** (and related **5A002, 5D002, 5E002**) classification, **License Exception ENC** (§ 740.17), **TSR**, **mass market** and **TSU**, and when **BIS/NSA notification** or **annual self-classification report** is needed. Prep only; no filing.

**Outputs:** Encryption Classification and Notification Memo (classification, ENC/TSR, notification and report obligations).

**Inputs:** Product/item description (encryption function, hardware/software), current or proposed ECCN, destination or "general", prior BIS registration/report if known.

**Why use it:** Encryption is a major pain point; a single memo tying CCL + ENC + notification in one place supports SaaS/hardware with crypto. Free to use; optional donation.
