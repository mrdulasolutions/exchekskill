# ExChek End-use / End-user Red Flag Assessment

Runs the BIS "Know Your Customer" red-flag checklist (Supplement No. 3 to 15 C.F.R. Part 732) for a given party or transaction and produces an auditable **red-flag assessment note** (no / yes / conditional; escalate if needed). Complements risk triage and screening with a dedicated end-use/end-user review for sales and compliance.
